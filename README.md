# Ashes and Demons

A top-down survival game in the browser. Fight off waves of demons, unlock weapons, and take on the boss.

**▶ Play it here: https://aimjnd.github.io/ashes-and-demons/**

Built with vanilla JavaScript, HTML canvas, and CSS — no engine, no frameworks, no build step.

## Run locally

Any static file server works, e.g.:

```
npx serve
```

then open the printed URL. (Opening `index.html` directly won't work — the game uses ES modules, which browsers block over `file://`.)
